# 🕸️ Crypto Intelligence HUB: Operational Pipeline (v1)

This reference guide outlines the "Skeleton Architecture" for a 24/7 autonomous crypto trading and development operation. 

---

## **🕷️ 1. THE SPIDER BUBBLE (Mental Model)**
Each Notion page acts as a specialized "Checkpoint" where agents operate independently and the Admin (User) oversees the output.

*   **CORE:** **24/7 Intelligence Hub (Admin Oversight)**
*   **BUBBLE 1: THE COLLECTIVE BRAIN (Brainstorm Operation)**
    *   24/7 ideation for Crypto Tools, TradingView Charts, and Code Updates.
*   **BUBBLE 2: THE MARKET SENTINEL (Autonomous Data)**
    *   24/7 scanning of On-Chain Flows, Wallet Tracking, and Derivatives Trends.
*   **BUBBLE 3: THE TOOL ARCHITECT (Development)**
    *   Implementing approved ideas into functional indicators and automated scripts.
*   **BUBBLE 4: THE ALERT SYNC (Communication)**
    *   Confirming signals between Scouts and Bookmap to trigger Slack alerts.

---

## **🏗️ 2. THE CREATOR PIPE (Tool Development)**
Autonomous 24/7 "living" development loop.

*   **INTAKE (Ideas/Pain Points):** **Notion `Brainstorm Session`**
    *   Agents propose new tools, chart setups, and code updates.
*   **THE LIVING DRAFT (Refinement):** **Notion `Strategy Wiki`**
    *   Technical requirements (Logic/Data/UI) are mapped after Admin "OK."
*   **PRODUCTION LINE (Execution):** **Notion `Goals & Tasks`**
    *   Agents break the draft into PineScript/Python coding tasks.

---

## **🛰️ 3. THE SENTINEL PIPE (Wallet & Coin Tracking)**
Autonomous 24/7 market monitoring and trend prediction.

*   **RAW FEED (24/7 Scanning):** **Notion `Master Data Log`**
    *   All raw scouts (Whale-Hunter/Sentinel) dump findings here.
*   **SENTINEL EVALUATOR (Tracking Trends):** **Notion `Asset Library` (Watchlist)**
    *   Agents update the "Health" and "Potential" status of each coin over time.
*   **ALERT GATEWAY (Real-Time Communication):** **Slack `#signals`**
    *   Priority alerts triggers when "High Potential" coins hit immediate momentum.

---

## **🔌 4. THE UNIVERSAL PIPELINE (Data Flow)**
A modular "Plug-and-Play" system where any data source can be ingested and filtered automatically.

1.  **DATA SINK (Notion `Master Data Log`):** The universal intake for ALL raw data (Glassnode, Hive, etc.).
2.  **TRAFFIC CONTROLLER (The Filter):** A script that "sorts" data based on confidence level into Brainstorm, Watchlist, or Issues.
3.  **HUMAN GATEKEEPER (The Approval Hub):** The Admin reviews brainstormed ideas and switches status to `OK` to trigger Goal creation.
4.  **TASK ENGINE (The Action Hub):** Automatically breaks approved Goals into daily actionable Tasks.

---

## **🤖 5. AGENT SKILL MATRIX (The Brains)**
Assigning specialized personas to subagents to improve brainstorming quality.

*   🐋 **The Forensic Whale-Hunter:** (On-Chain Forensic Analyst, Wallet Clustering).
*   📈 **The Derivatives Quant:** (Funding/OI Specialist, Liquidation Analyst).
*   💻 **The Indicator Architect:** (PineScript/Python Specialist, UI/UX Designer).

---

## **🖥️ 6. COMPUTE OPTIONS (Local vs. Cloud)**
*   **OPTION A (Railway Pro):** 24/7 Uptime. Agents run independently in the cloud. Alerts functional even with Local PC offline.
*   **OPTION B (Local PC):** High Performance/Zero Cost. Runs only when the PC is awake. Ideal for Bookmap data reading.
